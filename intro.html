标签管理

一目了然管理浏览器标签页，智能检测重复，告别标签混乱

你有没有遇到过这些情况？

开了太多标签，找不到想要的页面

浏览器标签栏挤满了小图标，想找某个页面只能一个个点开看，既费时又打断思路。

重复打开的页面，需要自己一个个判断

同一个网址不小心开了好几次，想清理却不知道哪些是重复的，手动对比 URL 既慢又容易漏。

来不及看的页面，想稍后再读

打开了很多有价值的文章但没时间细看，标签一直挂着占位置，关掉又怕忘记，两难。

核心功能

卡片式总览

所有标签按窗口分组，以卡片流形式展示。标题、网址、图标一目了然，点击即可跳转。

智能重复检测

自动识别窗口内重复和跨窗口重复，用不同底色区分。一键关闭窗口内重复项，保持浏览器整洁。

待办收藏

暂时不看的标签存入待办列表，自动关闭标签页。需要时一键重新打开，支持标记完成和删除。

安装方法

1

下载项目代码

选择其中一种方式即可：

直接下载 ZIP 推荐
  • 前往 GitHub 项目页面
  • 点击绿色的 Code 按钮
  • 选择 Download ZIP
  • 解压到任意位置
需要外网访问 GitHub。如无外网，请使用右侧 Git 克隆方式。
Git 克隆 方便更新

打开终端（macOS: ⌘+空格 输入 Terminal；Windows: Win+R 输入 cmd），执行：

$ git clone https://github.com/yixin68/tab-manager.git
$ cd tab-manager
2

打开 Chrome 扩展管理页

在 Chrome 浏览器中：

  • 点击浏览器右上角拼图图标，在弹出菜单底部点击「管理扩展程序」
3

开启开发者模式

在扩展管理页面的右上角，找到「开发者模式」开关，将其打开。开启后，页面顶部会出现一行新的操作按钮。

Chrome 不支持直接安装 .crx 扩展包，开发者模式是加载本地扩展的唯一方式。此模式仅影响你手动加载的扩展，不会影响 Chrome 的安全性。
4

加载扩展

  • 点击页面左上角出现的「加载已解压的扩展程序」按钮
  • 在弹出的文件选择器中，找到并选中你刚才克隆或解压的 tab-manager 文件夹（即包含 manifest.json 的那个目录）
  • 点击「选择文件夹」，Chrome 会自动加载扩展
加载成功后，Chrome 工具栏会出现扩展图标。如果没看到，点击工具栏右侧的拼图图标，将「标签管理」固定到工具栏即可。

使用指南

1

打开标签管理

点击浏览器工具栏中的扩展图标，在弹窗中可以看到当前标签数和重复数。点击弹窗中的「打开标签管理」按钮，进入完整的管理页面。

2

浏览与跳转

标签页按窗口分组展示为卡片流。点击卡片上的标题区域即可跳转到对应标签页，浏览器会自动切换到该窗口并激活标签。

3

清理重复项

当窗口内存在重复标签时，窗口标题旁会出现「关闭重复项」按钮，点击即可一键清理，保留第一条，关闭其余重复标签。暖黄色底色表示窗口内重复，蓝灰色底色表示跨窗口重复。

4

存为待办

悬停卡片时出现操作按钮，点击书签图标可将标签页存入待办列表并自动关闭该标签。点击右上角「待办」按钮打开待办抽屉，可重新打开、标记完成或删除待办项。

5

搜索过滤

在顶部搜索框输入关键词，即时过滤匹配的标签页，支持按标题和网址搜索。

6

关闭标签页

悬停卡片时点击关闭图标即可关闭标签页，会伴随花朵绽放动效。

键盘快捷键

Enter 跳转到标签页
Delete 关闭标签页
S 存为待办
